📖 遊び方・ルール

🏆 ゲームの目的

指定のラウンド数終了時に、最も多くの資産（P: ポイント）を持っていたプレイヤーが優勝！
資産＝「所持P ＋ 陣地の価値 ＋ 缶・ゴミの価値」の合計です。

🎮 ターンの流れ

  • ① サイコロ: 出た目の分「AP(行動力)」を獲得。
  • ② 移動: 1マス進むのに 1AP 消費(雨の日は2AP)。
  • ③ アクション: 止まったマスでAPを消費して行動。
  • ④ ターン終了: APが切れるか、やることが無くなったら「ターン終了」。

📊 ステータス

  • ⚡AP: 行動力。移動やアクションに必要。ターン終了で0に。
  • ❤️HP: 体力。0になると病院送りになり、所持Pが半減＆装備全ロスト。
  • 💰P: お金。アイテムの購入などに使用。

🎭 キャラクター固有能力

  • 🏃元アスリート: 移動コストが常に1AP。雨の日のペナルティ(移動2AP)を受けません。
  • 💼元営業マン: 話術を活かし、バイトの成功率が通常の50%から70%にアップします。
  • 🌿サバイバー: 危機察知能力が高く、ゴミ漁りでハズレを引いても警察に見つかるペナルティを無効化します。
  • 👊元ヤン: 他のプレイヤーと同じマスに止まると、自動的に相手から1Pをカツアゲ(強奪)します。

🗺️ マスの種類と効果

  • 🏥 病院(開始地点): HPが0になるとここへ搬送され、所持Pが半減します。
  • 🛣️ 通常の道: 特別な効果はありません。
  • 🥫 空き缶: 1APで缶を拾います(1ターン3回まで)。※雨の日は「雨具」が必要。
  • 🗑️ ゴミ山: 2APでゴミを漁ります。ハズレだと警察に見つかり次回AP-2！
  • 💱 買取所: 拾った空き缶とゴミを、現在の相場でPに換金します(0AP)。
  • 💼 バイト: 4APで挑戦。確率で成功すると10P獲得！
  • 🛒 ショップ: アイテムカードの購入や、手札の売却(2P)ができます。
  • 🎲 イベント: ミニゲームやストーリーが発生！Pやカードを獲得するチャンス。
  • 🏕️ 避難所: 止まると「ステルス」状態になり、次の敵を1回やり過ごせます。
  • 🕳️ マンホール: 1APで、別のマンホールマスへランダムにワープします。
  • 🚓 交番: 職務質問で足止めされ、そのターンはそれ以上移動できなくなります。
  • 🎯 目的地(的): 到達すると多額のボーナスPを獲得！的は別の場所へ移動します。
  • 🚩 陣地占領: 病院・イベント・ショップ・交番・マンホール【以外】のマスは、3APを消費して「自分の陣地」にできます。陣地からは毎ターン収入が入ります！

⚠️ NPC（敵）と危険なイベント

マップ上のNPCと同じマスに止まるか、ラウンド終了時のイベントで被害を受けます。「身代わり人形」を装備していると1回だけ無効化できます。

  • 🚓警察: 補導されると次回AP-2（身分証・ステルス等で回避可）※偶数ラウンドはパトロールでマップを動きます。
  • 😎ヤクザ: 30ダメージを受け、手札からカードをランダムに1枚奪われます。
  • 💀闇金: 強制的に所持金から10Pを没収されます。
  • 🧓厄介なおじさん: 絡まれてターンが強制終了し、カードを1枚失います。
  • 🤝仲間のホームレス: （※例外）出会うと空き缶を1つ分けてくれる有益なNPCです。
  • 🛻ごみ収集車(毎ラウンド終了時): ランダムにマップを暴走！轢かれると80の大ダメージ（一定確率でギリギリ回避）。

🃏 全カード一覧と効果

カードは自分のターンに2APで使用可能。手札上限は7枚（リュック装備で+2）です。

【バフ・回復系】

  • 支援面談: +5P＆次回AP+2
  • 炊き出し / 野良猫の導き: +3P確定 / +2P確定
  • 今日の運勢 / 密かなバイト: 50%でPが増えるか減る(ギャンブル)
  • 宝くじ当選: 10%で+15Pの大当たり
  • エナジードリンク / スケボー: 即座にAP+5 / 次のターンのダイス結果に+5AP

【装備品 (永続・使い切り)】

  • ボロボロの自転車: 毎ターン獲得AP+2
  • 安全靴: ゴミ漁りの消費APが1になる
  • 大きなリヤカー: 自分の陣地からの収入が2倍になる
  • リュック: 手札の上限が2枚増える
  • 雨具ゲット: 雨の日の移動・漁り不可ペナルティを無効化
  • 段ボールの盾 / ヘルメット: 次に受けるダメージを半減 / 完全無効化(1回で消費)
  • 身代わり人形: NPCや収集車などの妨害を1回だけ無効化(消費)

【攻撃・妨害系】

  • ステルス行動: 次に遭遇する敵・NPCを回避
  • 身分証明書: +1Pし、次に警察に捕まるのを防ぐ
  • 通報: 他のプレイヤー1人に次回AP-2のペナルティ
  • 缶泥棒: 他人から最大2Pを奪う
  • 領土挑戦状: サイコロで4以上が出れば他人の陣地を乗っ取る
  • レンガ / バット / 釘バット: 射程1〜2の他人に10〜30のダメージ
  • 拳銃 / パンチングマシン: 40 / 80(同マスのみ)の大ダメージ
  • ショットガン / レーザー銃: 射程内の【全員】に50 / 100の超ダメージ

【強力な全体効果 (逆転用)】

  • 大暴落: 全員の所持Pが半分になる
  • 下剋上: 1位のプレイヤーと自分の所持Pをそっくり入れ替える！
