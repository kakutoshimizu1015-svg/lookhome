🏠 脱・ホームレス：サバイバルシティ - ゲームルール

🎯 ゲームの目的（勝利条件）

ストリートで生き抜き、**「20P（ポイント）」を集めてマップ左下にある「更生支援センター」**のマスに一番最初に到達したプレイヤーが勝利となります。

👥 プレイ人数

1〜4人（CPUを混ぜて遊ぶことも可能です）

オンライン対戦・オフライン（1台のスマホ/PCで回しプレイ）両対応

🎲 基本となるターン進行

自分のターンが回ってくると、以下の順序で進行します。

ドロー: ターン開始時にカードを自動で1枚引きます。（手札の初期上限は7枚）

サイコロを振る: 「🎲 サイコロを振る」ボタンを押して、2つのサイコロを振ります。出目の合計が、このターンに使える**「AP（アクションポイント）」**になります。

アクション実行: 獲得したAPを消費して、「移動」や「資源回収」、「カードの使用」などのアクションを行います。APが許す限り何度でも行動できます。

ターン終了: APが0になるか、やることがなくなったら「🛑 ターン終了」ボタンを押して次のプレイヤーに手番を回します。

※手札上限オーバーに注意: 手札が上限を超えているとターンを終了できません。カードを使用するか捨ててください。

💰 リソース

⚡ AP (アクションポイント): 行動するための体力です。サイコロで獲得し、ターン終了時にリセットされます。

💰 P (ポイント/お金): 勝利に必要な資産です。空き缶拾いやバイト、陣地収入などで増やします。20P集めるのが目標です。

🏃 アクション一覧

獲得したAPを消費して以下の行動ができます。

🚶 移動 (1AP / 雨天時2AP): 隣接するマスへ移動します。

🥫 空き缶拾い (1AP): 空き缶マス限定。確実に +1P を獲得します。（1ターンに3回まで）

🗑️ ゴミ漁り (2AP): ゴミ山マス限定。ランダムでPを獲得しますが、失敗すると「警察」に見つかり、次回AP-2のペナルティを受けます。

🚩 陣地占領 (3AP): 現在のマスを自分の「領土」にします。以降、自分のターン開始時に領土の数だけボーナスPが入ります。（おじさんがいるマスは占領不可）

💼 小バイト (4AP): 50%の確率で成功し、+5P を獲得します。

🎴 カード使用 (2AP): 手札からアイテムやイベントカードを使用します。

🌙 ラウンド終了イベントとNPC (お邪魔キャラ)

全員のターンが1周終わると「ラウンド」が終了し、ストリートに様々な変化が起こります。

🌤️ 天候と環境の変化

雨 (🌧️): 20%の確率で雨が降ります。雨の間は移動コストが2APになり、「空き缶」「ゴミ漁り」ができなくなります（「雨具」カードで無効化可能）。

道路工事 (🚧): ランダムなマスが工事中になり、2ラウンドの間そこを通行できなくなります。

⚠️ NPCの移動と妨害

NPCと同じマスに止まったり、ラウンド終了時にNPCが移動してきて接触すると悪いことが起こります。

👮‍♂️ 警察官: 接触すると職務質問を受け、ターンが強制終了し次回AP-2のペナルティ。身分証やステルス状態ならやり過ごせます。

🧓 厄介なおじさん: 彼の縄張り（マス）に入ってしまうと絡まれ、**カードを1枚失い、ターンが強制終了（AP全没収）**します。

🐀 野良動物: 資源マスをうろつきます。動物がいるマスでは「空き缶」「ゴミ漁り」がブロックされて実行できません。

🛻 ゴミ収集車: ラウンド終了時にマップを爆走します。轢かれると所持Pが半分になってしまいます。ステルス状態か、運良く避けられるのを祈りましょう。
